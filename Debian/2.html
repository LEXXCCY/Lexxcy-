Perintah Dasar Linux

Selamat datang di tutorial dasar Linux. Di sini, kita akan membahas beberapa perintah dasar yang sering digunakan di sistem operasi Linux.

1. ls

Perintah ls digunakan untuk menampilkan daftar file dan direktori di dalam direktori saat ini. Anda bisa menggunakan opsi tambahan seperti -l untuk menampilkan detail lebih lanjut, atau -a untuk menampilkan file tersembunyi.

ls

2. cd

Perintah cd digunakan untuk berpindah direktori. Misalnya, cd /home/user/Documents akan memindahkan Anda ke direktori Documents di dalam direktori user.

3. cp

Perintah cp digunakan untuk menyalin file atau direktori. Anda perlu menentukan file sumber dan tujuan. Untuk menyalin direktori, gunakan opsi -r.

cp file1.txt /home/user/Documents/
cp -r dir1 /home/user/

4. mv

Perintah mv digunakan untuk memindahkan atau mengganti nama file dan direktori. Misalnya, mv file1.txt /home/user/ akan memindahkan file1.txt ke direktori user.

5. rm

Perintah rm digunakan untuk menghapus file atau direktori. Gunakan opsi -r untuk menghapus direktori beserta isinya. Berhati-hatilah saat menggunakan perintah ini, karena file yang dihapus tidak dapat dikembalikan.

rm file1.txt
rm -r dir1

6. mkdir

Perintah mkdir digunakan untuk membuat direktori baru. Anda dapat membuat beberapa direktori sekaligus dengan menuliskan nama-nama direktori yang dipisahkan oleh spasi.

mkdir newdir
mkdir dir1 dir2 dir3

7. nano

Perintah nano adalah editor teks berbasis terminal yang sederhana. Anda bisa menggunakannya untuk membuat atau mengedit file teks. Misalnya, untuk mengedit file1.php, jalankan:

nano file1.php

8. ping

Perintah ping digunakan untuk mengecek koneksi jaringan ke sebuah server atau domain. Ini mengirimkan paket data ke alamat yang ditentukan dan menampilkan waktu responsnya.

ping namadomain.com

Apabila namadomain.com dalam kondisi online, akan muncul respons dengan output time=xxx ms. Jika offline, akan muncul respons request timed out atau destination host unreachable.

9. apt

Perintah apt adalah manajer paket untuk sistem berbasis Debian, seperti Ubuntu. Anda bisa menggunakannya untuk menginstal, menghapus, dan memperbarui paket perangkat lunak.

sudo apt update
sudo apt install paketname

10. reboot

Perintah reboot digunakan untuk me-restart sistem Linux. Proses ini biasanya dilakukan setelah melakukan perubahan penting atau pembaruan sistem. Pastikan semua aplikasi penting sudah disimpan sebelum menjalankan perintah ini.

sudo reboot
