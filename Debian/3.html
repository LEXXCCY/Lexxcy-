Apache HTTP Webserver adalah salah satu web server yang paling banyak digunakan di dunia. Artikel ini akan membahas cara mengonfigurasi Apache HTTP Webserver di sistem operasi Debian.

1. Instalasi Apache

Untuk menginstal Apache di Debian, buka terminal dan jalankan perintah berikut:

sudo apt update
sudo apt install apache2

2. Memeriksa Status Apache

Untuk memeriksa status Apache, gunakan perintah ini:

sudo systemctl status apache2

3. Konfigurasi Virtual Host

Untuk menambahkan konfigurasi virtual host, buat file baru di direktori /etc/apache2/sites-available/. Misalnya:

sudo nano /etc/apache2/sites-available/example.com.conf

Isi file tersebut dengan konfigurasi berikut:

<VirtualHost *:80>
    ServerAdmin webmaster@example.com
    ServerName example.com
    DocumentRoot /var/www/example.com
    ErrorLog ${APACHE_LOG_DIR}/error.log
    CustomLog ${APACHE_LOG_DIR}/access.log combined
</VirtualHost>

4. Mengaktifkan Virtual Host

Aktifkan konfigurasi virtual host yang baru saja dibuat dengan perintah:

sudo a2ensite example.com.conf

Dan restart Apache untuk menerapkan perubahan:

sudo systemctl reload apache2

5. Kesimpulan

Anda telah berhasil mengonfigurasi Apache HTTP Webserver di Debian. Sekarang, Anda dapat mengakses situs web Anda menggunakan domain yang telah Anda konfigurasi.
